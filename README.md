# my-blog

基于hugo 的个人blog

# 使用

```bash
cd themes
git clone git@github.com:liseen315/bilberry-hugo-theme.git
git submodule add -f git@github.com:liseen315/liseen315.github.io.git public
```

# 预览

```bash
hugo server -D
```

# 编辑脚本

```bash
bash new.sh name
```

# 发布

```bash
hugo // 生成静态页
bash deploy.sh 推送到githubpage
```
