人生苦短,我学Python...Form表单用于提交资源信息,比如用户登录上传等操作,Flask提供了强大的表单创建配合Jinja模板引擎会大大加快应用的开发速度.

<!--more-->

文件目录结构如下:

```bash
├── app.py    # 入口
├── forms.py  # Form表单
├── static    # 静态资源目录
│   ├── css
│   ├── favicon.ico
│   ├── js
│   └── style.css
└── templates   # Jinja模板
    ├── base.html
    ├── bootstrap.html
    └── index.html
```

app.py

```python
from flask import Flask,render_template,redirect,url_for,flash
from forms import LoginForm
app = Flask(__name__)

app.secret_key = 'dev secret'

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/bootstrap',methods=['GET','POST'])
def bootstrap():
    form = LoginForm()
    if form.validate_on_submit():
        username = form.username.data
        flash('Welcom home, %s!' % username)
        return redirect(url_for('index'))
    return render_template('bootstrap.html',form=form)
```

forms.py

```python
from wtforms import StringField,PasswordField,BooleanField,SubmitField
from wtforms.validators import DataRequired,Length
from flask_wtf import FlaskForm

class LoginForm(FlaskForm):
    username = StringField('Username',validators=[DataRequired()])
    password = PasswordField('Password',validators=[DataRequired(),Length(8, 128)])
    remember = BooleanField('Remember me')
    submit = SubmitField('Log in')

```

bootstrap.html

```html
{% extends 'base.html' %}

{% block metas %}
    {{ super() }}
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
{% endblock %}

{% block styles %}
    <link rel="stylesheet" href="{{ url_for('static', filename='css/bootstrap.min.css') }}">
{% endblock %}

{% block nav %}{% endblock %}

{% block content %}
    <div class="container">
        <br>
        <a href="{{ url_for('index') }}">&larr; Return</a>
        <h1 class="display-4">Bootstrap Style Form</h1>
        <hr>
        <form method="post">
            {{ form.csrf_token }}
            <div class="form-group">
                {{ form.username.label }}
                {% if form.username.errors %}
                    {{ form.username(class='form-control is-invalid') }}
                    {% for error in form.username.errors %}
                        <div class="invalid-feedback">
                        {{ error }}
                        </div>
                    {% endfor %}
                {% else %}
                    {{ form.username(class='form-control') }}
                {% endif %}
            </div>
            <div class="form-group">
                {{ form.password.label }}
                {% if form.password.errors %}
                    {{ form.password(class='form-control is-invalid') }}
                    {% for error in form.password.errors %}
                        <div class="invalid-feedback">
                        {{ error }}
                        </div>
                    {% endfor %}
                {% else %}
                    {{ form.password(class='form-control') }}
                {% endif %}

            </div>
            <div class="form-check">
                {{ form.remember(class='form-check-input') }}
                {{ form.remember.label }}
            </div>
            {{ form.submit(class='btn btn-primary') }}
        </form>
    </div>
{% endblock %}

{% block footer %}{% endblock %}

```

至此基础的带验证的表单已经创建好了.当点Submit的时候会进行服务端的基础验证,如果验证通过会返回到首页

Tips:之前我们在JinJa2的模板语法内使用过宏,当有大量的表单需要进行重复的编写类似模板的时候可以使用宏来节省大量的时间.
Flask也支持自定义验证.本地化验证等功能.
