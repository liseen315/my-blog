很多情况下站点都需要登录,没登录情况下跳转到登录页面,登录后进到指定页面,这时候就应用到了重定向功能

```python
@app.route('/login/')
def login():
    return  '这是登录界面'

# 页面重定向这里暂时用参数进行模拟,如果参数=1就进到指定页面,如果不是,就跳转到登录页面
@app.route('/question/<islogin>')
def question(islogin):
    if islogin == '1':
        return '这是发布问答页面'
    else:
        redirect_url = url_for('login')
        return  redirect(redirect_url)
```
